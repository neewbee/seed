# 前端开发种子工程

## 主要框架
* [antd] - HTML enhanced for web apps!
* [dva] - awesome web-based text editor
* [bookshelf] - ORM
* [knex] - query string builder
* [Express] - fast node.js network app framework [@tjholowaychuk]

----

## Getting Started
Install dependencies.

```bash
$ npm install
```

Start server.

```bash
$ npm run dev
```
